ftmetric-helm

Helm repository for ftmetric, a Helm chart for deploying ftmetric component from fthelper project.

Usage

Add this repository to your Helm:

helm repo add ftmetric-helm https://n0rmanc.github.io/ftmetric-helm
helm repo update

Install the chart:

helm install my-release ftmetric-helm/ftmetric
